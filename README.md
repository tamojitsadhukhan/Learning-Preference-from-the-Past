# Learning Preferences from the Past
# Simulation Results and Real Data Analysis

This repository contains the R code for the simulation experiments and real data analysis in [1]. All figures and tables in [1] are reproducible using the initial seeds provided in the code.

## Simulation Results

Code for performing the simulation experiments in [1] is provided in the following notebook:

  - `simulation.ipynb`

## Real Data Analysis

Code for analysing the MovieLens 20M Dataset, as depicted in [1], is provided in the following notebook:

- `real-data.ipynb`

## Data Source

The MovieLens 20M Dataset is publicly available at [MovieLens 20M Dataset](https://grouplens.org/datasets/movielens/20m/) [2].

For our analysis, we use the version available on the Kaggle platform: [MovieLens 20M Dataset](https://www.kaggle.com/datasets/grouplens/movielens-20m-dataset).

## Reproducibility

The analysis was conducted using the Kaggle platform. The [MovieLens 20M Dataset](https://www.kaggle.com/datasets/grouplens/movielens-20m-dataset) was accessed by adding it as an input dataset within the Kaggle notebook environment using the ***Add Input*** functionality. To reproduce the results:

1. Open a Kaggle Notebook
2. Click **"Add Input"**
3. Search for: *MovieLens 20M Dataset*
4. Add the dataset to the notebook
5. Run the code
